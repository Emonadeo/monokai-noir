# Monokai Pro for JetBrains

The unofficial Monokai Pro theme, adapted for JetBrains by Thematic. This plugin includes the default theme and all filters: Machine, Octagon, Ristretto, and Spectrum. Credit goes to the original creator: [https://monokai.pro](https://monokai.pro)
